ChatServer API 接口文档

仿微信即时通讯后端 — Express + MySQL + Socket.io

Base URL: http://localhost:3000

接口目录

RSA 加密 用户系统 好友系统 消息系统 群聊系统 WebSocket 事件 数据库设计
通用说明

响应格式：所有接口返回 JSON，成功格式为 { code: 200, msg: "...", data: {...} }，失败为 { code: 4xx/5xx, msg: "..." }

鉴权方式：需要登录的接口在请求头添加 Authorization: Bearer <token>

Content-Type：JSON 请求使用 application/json，文件上传使用 multipart/form-data

RSA 加密
GET /rsa/public-key

获取 RSA 公钥，用于前端加密密码

{ "publicKey": "-----BEGIN PUBLIC KEY-----\nMIIBIjANBg..." }
POST /rsa/decrypt

RSA 解密接口，传入加密后的 Base64 密文

请求参数 (Body)

参数类型必填说明
encryptedstring是RSA 加密后的 Base64 字符串
{ "decrypted": "123456" }
用户系统
GET /user/captcha

获取验证码图片（SVG 格式），有效期 5 分钟

{ "id": "m1a2b3c4d", "svg": "<svg ...>...</svg>" }
POST /user/register

用户注册（multipart/form-data）

请求参数 (Body)

参数类型必填说明
usernamestring是用户名（唯一）
encryptedstring是RSA 加密后的密码
nicknamestring否昵称，默认为用户名
captchaIdstring是验证码 ID
captchaCodestring是验证码
avatarfile否头像图片，默认使用默认头像
POST /user/login

用户登录，成功返回 JWT token

请求参数 (Body - JSON)

参数类型必填说明
usernamestring是用户名
encryptedstring是RSA 加密后的密码
captchaIdstring是验证码 ID
captchaCodestring是验证码
{ "code": 200, "msg": "登录成功", "data": { "token": "eyJhbGciOiJIUzI1NiIs...", "user": { "id": 1, "username": "admin", "nickname": "管理员", "avatar": "/uploads/xxx.png", "avatarUrl": "http://localhost:3000/uploads/xxx.png" } } }
GET /user/search 需要登录

按用户名或昵称模糊搜索用户，排除自己，最多返回 20 条

请求参数 (Query)

参数类型必填说明
keywordstring是搜索关键字
{ "code": 200, "data": [ { "id": 2, "username": "admin", "nickname": "管理员", "avatar": "/uploads/xxx.png" } ] }
GET /user/profile 需要登录

获取当前登录用户的资料

{ "code": 200, "data": { "id": 1, "username": "admin", "nickname": "管理员", "avatar": "/uploads/xxx.png", "avatarUrl": "...", "created_at": "...", "last_online": "..." } }
PUT /user/profile 需要登录

修改当前用户昵称

请求参数 (Body - JSON)

参数类型必填说明
nicknamestring是新昵称
POST /user/avatar 需要登录

修改当前用户头像（multipart/form-data）

请求参数 (Body)

参数类型必填说明
avatarfile是头像图片文件
{ "code": 200, "msg": "头像修改成功", "data": { "avatarUrl": "http://localhost:3000/uploads/chat_xxx.png" } }
好友系统
POST /friend/request 需要登录

发送好友申请（不能申请自己、不能重复申请、已是好友则拒绝）

请求参数 (Body - JSON)

参数类型必填说明
friendIdnumber是目标用户 ID
messagestring否申请验证消息
POST /friend/accept 需要登录

接受好友申请，自动建立双向好友关系

请求参数 (Body - JSON)

参数类型必填说明
requestIdnumber是好友申请 ID
POST /friend/reject 需要登录

拒绝好友申请

请求参数 (Body - JSON)

参数类型必填说明
requestIdnumber是好友申请 ID
GET /friend/requests 需要登录

获取收到的待处理好友申请列表

{ "code": 200, "data": [ { "id": 1, "from_user_id": 3, "message": "你好", "created_at": "...", "username": "test", "nickname": "测试用户", "avatar": "..." } ] }
GET /friend/list 需要登录

获取好友列表，每个好友包含未读消息数

{ "code": 200, "data": [ { "id": 2, "username": "admin", "nickname": "管理员", "avatar": "...", "last_online": "...", "unread_count": 3 } ] }
POST /friend/delete 需要登录

删除好友（双向解除关系）

请求参数 (Body - JSON)

参数类型必填说明
friendIdnumber是要删除的好友 ID
消息系统
GET /message/history 需要登录

获取与某好友的聊天记录（分页，按时间正序）

请求参数 (Query)

参数类型必填说明
friendIdnumber是好友用户 ID
pagenumber否页码，默认 1
sizenumber否每页条数，默认 20
{ "code": 200, "data": [ { "id": 1, "sender_id": 1, "receiver_id": 2, "content": "你好", "type": "text", "is_read": 1, "is_recalled": 0, "created_at": "..." } ] }
GET /message/unread 需要登录

获取未读消息统计（按发送者分组）

{ "code": 200, "data": [ { "sender_id": 2, "count": 5 }, { "sender_id": 3, "count": 2 } ] }
POST /message/recall 需要登录

撤回已发送的消息（仅发送者可操作，2 分钟内有效）

请求参数 (Body - JSON)

参数类型必填说明
messageIdnumber是消息 ID
{ "code": 200, "msg": "撤回成功", "data": { "receiverId": 2 } }
POST /message/upload 需要登录

上传聊天文件/图片（支持图片、PDF、Office、压缩包等，最大 10MB）

请求参数 (Body - multipart/form-data)

参数类型必填说明
filefile是文件
{ "code": 200, "data": { "url": "http://localhost:3000/uploads/chat_123456.png", "type": "image", "filename": "photo.png", "size": 102400 } }

参数类型必填说明

POST /group/quit 需要登录

参数类型必填说明

参数类型必填说明

POST /group/disband 需要登录

解散群聊（仅群主可操作，级联删除群消息和成员）

请求参数 (Body - JSON)

参数类型必填说明

GET /group/history 需要登录

获取群聊天记录（分页）

请求参数 (Query)

参数类型必填说明
groupIdnumber是群 ID
pagenumber否页码，默认 1
sizenumber否每页条数，默认 20
POST /group/recall 需要登录

撤回群消息（仅发送者，2 分钟内有效）

请求参数 (Body - JSON)

参数类型必填说明
messageIdnumber是消息 ID
{ "code": 200, "msg": "撤回成功", "data": { "groupId": 1 } }
WebSocket 事件
连接地址: ws://localhost:3000  |  认证方式: { auth: { token: "Bearer xxx" } }
private_message 双向

私聊消息收发。客户端发送 { receiverId, content, type }，服务端推送给接收者和发送者回执。

// 发送 socket.emit('private_message', { receiverId: 2, content: '你好', type: 'text' }); // 接收（type 可选: text | image | file） socket.on('private_message', (msg) => { // msg: { id, senderId, receiverId, content, type, createdAt } });
group_message 双向

群聊消息收发。客户端发送 { groupId, content, type }，服务端广播给群内所有成员。

socket.emit('group_message', { groupId: 1, content: '大家好', type: 'text' }); socket.on('group_message', (msg) => { // msg: { id, groupId, senderId, content, type, createdAt } });
message_read 客户端 → 服务端

标记消息已读。发送 { senderId }，将该用户发来的所有未读消息标记为已读。

messages_read 服务端 → 客户端

通知发送者消息已被对方读取。{ readerId }

message_recall 双向

私聊消息撤回通知。客户端发送 { messageId, receiverId }，服务端推送给接收者 { messageId, senderId }。

group_message_recall 双向

群消息撤回通知。客户端发送 { messageId, groupId }，服务端广播给群内所有成员 { messageId, senderId, groupId }。

online_users 服务端 → 客户端

连接成功后推送当前在线用户 ID 数组。[1, 2, 3]

user_online 服务端 → 客户端

广播某用户上线。{ userId }

user_offline 服务端 → 客户端

广播某用户下线。{ userId, lastOnline }

unread_messages 服务端 → 客户端

连接成功后推送未读消息，按发送者分组。{ senderId: [msg1, msg2, ...] }

数据库设计
users - 用户表
字段类型说明
idINT PK用户 ID
usernameVARCHAR(50) UNIQUE用户名
passwordVARCHAR(255)密码（明文存储，前端 RSA 加密传输）
nicknameVARCHAR(50)昵称
avatarVARCHAR(255)头像路径
created_atTIMESTAMP注册时间
last_onlineTIMESTAMP最后在线时间
friends - 好友关系表
字段类型说明
idINT PK记录 ID
user_idINT FK用户 ID
friend_idINT FK好友 ID
created_atTIMESTAMP建立关系时间
friend_requests - 好友申请表
字段类型说明
idINT PK申请 ID
from_user_idINT FK申请人 ID
to_user_idINT FK被申请人 ID
statusENUMpending / accepted / rejected
messageVARCHAR(200)申请验证消息
created_atTIMESTAMP申请时间
messages - 私聊消息表
字段类型说明
idINT PK消息 ID
sender_idINT FK发送者 ID
receiver_idINT FK接收者 ID
contentTEXT消息内容（文字/图片URL/文件URL）
typeENUMtext / image / file
is_readTINYINT(1)是否已读
is_recalledTINYINT(1)是否已撤回
created_atTIMESTAMP发送时间
user_groups - 群表
字段类型说明
idINT PK群 ID
nameVARCHAR(100)群名称
owner_idINT群主 ID
created_atTIMESTAMP创建时间
group_members - 群成员表
字段类型说明
group_idINT FK (PK)群 ID
user_idINT FK (PK)用户 ID
joined_atTIMESTAMP加入时间
group_messages - 群消息表
字段类型说明
idINT PK消息 ID
group_idINT FK群 ID
sender_idINT FK发送者 ID
contentTEXT消息内容
typeENUMtext / image / file
is_recalledTINYINT(1)是否已撤回
created_atTIMESTAMP发送时间
